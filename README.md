# Convite para Alanis — GitHub Pages

Esta pasta já está pronta para publicação no GitHub Pages.

1. Envie `index.html`, `style.css` e `script.js` para a raiz do repositório.
2. No GitHub, abra **Settings → Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Escolha a branch `main`, a pasta `/ (root)` e clique em **Save**.

A música é reproduzida por um player incorporado do vídeo oficial de **Ironic**, de Alanis Morissette. Como os navegadores bloqueiam áudio automático, ela começa somente depois do clique em **Começar a sessão**.
